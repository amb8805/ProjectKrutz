SETTING ManifestExtractor.xml
-----------------------------

1. Copy the script the to relative root folder of .apk folder hierarchy.

2. Open a Terminal console.

3. Move to the folder where the bash script is located using the command cd. For more 
information about the cd command type 'man cd' in the console without quotes.

4. Proceed to execute the command by typing 'bash ManifestExtractor.xml' without quotes.
